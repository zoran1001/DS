# 静态网站开发必备VS Code扩展

以下是适合HTML、CSS、JavaScript静态网站开发的VS Code扩展，按功能分类整理：

## 1. 基础编辑辅助

### Chinese (Simplified) Language Pack
- **功能**：将VS Code界面汉化为中文
- **用途**：适合中文用户，降低使用门槛

### vscode-icons
- **功能**：为文件和文件夹提供丰富的图标
- **用途**：使资源管理器更加直观，提高文件识别效率

### Auto Rename Tag
- **功能**：自动重命名HTML/XML标签
- **用途**：修改标签时自动同步修改闭合标签，减少错误

### Auto Close Tag
- **功能**：自动添加HTML/XML闭合标签
- **用途**：提高HTML编写速度，避免标签不匹配错误

## 2. 代码格式化与美化

### Prettier - Code formatter
- **功能**：统一代码风格，支持HTML、CSS、JavaScript等多种语言
- **用途**：保持代码整洁一致，提高可读性和维护性
- **配置**：已在项目中配置 `.prettierrc` 文件

### HTML CSS Support
- **功能**：为HTML提供CSS类名和ID的智能提示
- **用途**：提高CSS类名和ID的编写效率，减少拼写错误

### CSS Peek
- **功能**：从HTML中快速查看和跳转到CSS定义
- **用途**：方便查看和修改CSS样式，提高开发效率

## 3. 实时预览与调试

### Live Server
- **功能**：启动本地开发服务器，支持实时刷新
- **用途**：修改代码后自动刷新浏览器，实时查看效果

### BrowserSync for VS Code
- **功能**：类似Live Server，支持多设备同步测试
- **用途**：在多个设备上同步预览网站，方便测试响应式设计

### Debugger for Chrome
- **功能**：在VS Code中调试JavaScript代码
- **用途**：方便地调试前端JavaScript代码，提高调试效率

## 4. 语法高亮与智能提示

### Bracket Pair Colorizer 2
- **功能**：为匹配的括号添加相同颜色
- **用途**：方便识别嵌套结构，减少语法错误

### Highlight Matching Tag
- **功能**：高亮显示匹配的HTML标签
- **用途**：方便查看HTML标签的嵌套关系

### Path Intellisense
- **功能**：提供文件路径的智能提示
- **用途**：快速输入文件路径，减少路径错误

## 5. 其他实用工具

### GitLens - Git supercharged
- **功能**：增强Git功能，显示代码作者、修改时间等信息
- **用途**：方便查看和管理Git版本控制

### TODO Highlight
- **功能**：高亮显示代码中的TODO、FIXME等注释
- **用途**：方便追踪待办事项和需要修复的问题

### Code Spell Checker
- **功能**：检查代码中的拼写错误
- **用途**：减少变量名、注释等中的拼写错误

## 6. 项目特定扩展

### html-minifier-terser
- **功能**：压缩HTML文件
- **用途**：减少文件体积，优化网站加载速度

### cssnano
- **功能**：压缩CSS文件
- **用途**：减少CSS文件体积，优化加载性能

## 如何安装扩展

1. 打开VS Code
2. 点击左侧边栏的扩展图标（或按 `Ctrl+Shift+X`）
3. 在搜索框中输入扩展名称
4. 点击"安装"按钮即可

## 推荐安装的扩展组合

对于静态网站开发，推荐至少安装以下扩展：
- Chinese (Simplified) Language Pack（中文界面）
- vscode-icons（文件图标）
- Prettier - Code formatter（代码格式化）
- Auto Rename Tag + Auto Close Tag（标签辅助）
- Live Server（实时预览）
- HTML CSS Support + CSS Peek（CSS辅助）

这些扩展将大大提升您的开发效率，使静态网站开发更加顺畅。
